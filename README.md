# Civ6 Lobby Monitor (Civ6banalert)

Sid Meier's Civilization VI のマルチプレイヤーロビーを監視し、特定のプレイヤー（警戒対象など）が入室した際に音と画面表示で通知するツールです。

## 機能

- **リアルタイム監視**: Civ6の接続ログを監視し、プレイヤーの入室・退室を検知します。
- **アラート通知**: 警戒リスト（`watchlist.txt`）に登録されたプレイヤーが入室すると、警告音と赤文字で通知します。
- **リスト管理**: 専用ツールを使用して、Steam IDやプロフィールURLから簡単に警戒対象を追加できます。

## ファイル構成

- `civ6_watchv2.py`: 監視メインスクリプト。
- `add_player.py`: 警戒リスト（`watchlist.txt`）への追加ツール。
- `start_civ6_watch.bat`: 監視を簡単に開始するためのバッチファイル。
- `watchlist.txt`: 警戒対象のリスト（自動生成されます）。

## 使い方

### 1. 準備
Pythonがインストールされている必要があります。

### 2. 監視の開始
`start_civ6_watch.bat` をダブルクリックして実行してください。
Civ6が起動し、ログが出力され始めると監視が開始されます。

### 3. 警戒対象の追加
`add_player.py` を実行してください。
1. Steam ID（17桁の数字）またはプロフィールのURLを入力します。
2. プレイヤー名（任意）を入力します。
3. 理由（任意）を入力します。
4. `watchlist.txt` に保存されます。
※監視中の場合は、スクリプトを再起動すると反映されます。

## 注意事項
- このツールは `net_connection_debug.log` を参照します。
- Civ6の設定や環境によってはログの出力場所が異なる場合があります。
- 音を鳴らすために `winsound` モジュール（Windows標準）を使用しています。
